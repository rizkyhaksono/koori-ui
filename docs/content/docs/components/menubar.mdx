## Usage

```tsx
import {
  GlassMenubar,
  GlassMenubarMenu,
  GlassMenubarTrigger,
  GlassMenubarContent,
  GlassMenubarItem,
  GlassMenubarSeparator,
} from "koori-ui";

<GlassMenubar>
  <GlassMenubarMenu>
    <GlassMenubarTrigger>File</GlassMenubarTrigger>
    <GlassMenubarContent>
      <GlassMenubarItem>New Tab</GlassMenubarItem>
      <GlassMenubarSeparator />
      <GlassMenubarItem>Print‚Ä¶</GlassMenubarItem>
    </GlassMenubarContent>
  </GlassMenubarMenu>
</GlassMenubar>
```

## Props

### GlassMenubar

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string` | ‚Äî | Controlled active menu |
| `defaultValue` | `string` | ‚Äî | Default active menu |
| `onValueChange` | `(value: string) => void` | ‚Äî | Fired when active menu changes |
| `loop` | `boolean` | `true` | Whether keyboard navigation loops |

### GlassMenubarItem

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `disabled` | `boolean` | `false` | Disable the item |
| `onSelect` | `(event) => void` | ‚Äî | Fired when the item is selected |

All Radix Menubar props are also supported.
