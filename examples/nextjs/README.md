# Koori UI — Next.js Example

**Live demo: [https://koori-ui.vercel.app](https://koori-ui.vercel.app)**

This is a fully-featured demonstration of the Koori UI component library, built with Next.js. It showcases all 27+ components, theming capabilities, and color presets in action.

## Features

- 🪟 All Koori UI components with interactive examples
- 🌗 Dark/Light/System theme switching with `KooriProvider`
- 🎨 Live color scheme switching (Slate, Zinc, Neutral, Violet)
- 📱 Responsive design with Tailwind CSS
- ⚡ Built with Next.js 16 and React 19

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build & Deploy

```bash
npm run build
npm start
```

Deployed on [Vercel](https://vercel.com).

## Project Structure

```
app/
  layout.tsx        # KooriProvider setup
  page.tsx          # Component showcase
  globals.css
```
